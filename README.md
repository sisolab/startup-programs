K-Startup(창업진흥원)의 모집중 창업지원 공고 전체를 가져와 텍스트 형태로 제공하는 레포입니다.
사용자는 AI 프롬프트를 활용하여 이 데이터를 읽어와 아래와 같은 결과를 출력하도록 요청할 수 있습니다.
<img src="./docs/curation-example.jpeg" alt="큐레이션 결과 예시" width="600">

## 사용 방법

1. 아래 내용을 AI(ChatGPT, 클로드 등) 채팅창에 입력합니다.
```
이 프롬프트 실행해줘. https://raw.githubusercontent.com/sisolab/startup-programs/main/kstartup-curation-prompt.md
```
2. AI가 내 프로필 내용을 물어봅니다. 예시를 보고 정보를 적어서 입력합니다. 자세한 정보를 적을수록 나에게 매칭되는 정보를 찾아줍니다.
3. 같은 채팅창에서 "프롬프트 다시 실행해줘", "창업 지원 프로그램 검색 다시 해 줘" 같이 입력하면 최신 공고로 재탐색됩니다.

## 주의사항
- 불러오는 데이터가 크기 때문에 AI 유료 사용량이 빠르게 소진될 수 있습니다. 일주일에 1번 정도 실행을 권장합니다. 
- AI 모델에 데이터를 해석하고 매칭하는 결과가 다릅니다. ChatGPT, Claude에서 테스트했으며 Claude Opus 4.7이 가장 좋은 결과를 보였습니다.
- 중요 내용은 반드시 **공식 공고에서 다시 확인**하세요.

## 활용 예시

한 번 큐레이션을 받은 뒤 같은 채팅창에서 다양한 조건으로 재탐색할 수 있습니다.

- "내 프로필 정보 업데이트 하고 싶어"
- "글로벌 진출 지원 사업 중심으로 나와 매치되는 공고만 다시 찾아줘"
- "내 정보와 관계없이 사업화자금 지원이 큰 순서대로 10개만 출력해줘"

## 라이선스 / 금지 사항

- 프롬프트의 **웹 업로드 금지** (사적 배포는 허용)
- 다른 서비스 제작을 위한 **소스로 사용 금지**
